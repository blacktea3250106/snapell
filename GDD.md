# VoxCards — 萬物皆卡，以聲召喚

## 完整遊戲設計文件（Game Design Document）

---

# 一、遊戲概述

## 一句話介紹

> **對著你的貓大喊一句中二咒語，牠就變成一張能打架的卡牌。**

## 核心概念

VoxCards 是一款「現實召喚」卡牌對戰遊戲。玩家用手機相機拍攝現實中的任何事物，對著它喊出一句咒語，AI 就會將畫面和聲音融合，煉成一張獨一無二的卡牌。然後用這些卡牌跟全世界的玩家對戰。

## 核心循環

```
📷 拍攝現實 → 🗣️ 喊出咒語 → 🌀 AI 煉成 → 🃏 卡片誕生 → ⚔️ 對戰 → 📷 繼續召喚...
```

## 為什麼會爆火

| 要素 | 怎麼做到的 |
|------|-----------|
| UGC 無限 | 每張卡都是玩家用現實 + 聲音創造的，永遠不重複 |
| 短影片友好 | 念咒語的畫面天生就是內容，搞笑 / 中二 / 感人都有 |
| 線下社交 | 共鳴召喚 + 地點限定 → 玩家會為了遊戲出門 |
| 低門檻高上限 | 拍照念咒語誰都會，但組合優化深不見底 |
| 開盲盒心理 | 每次召喚都是賭品質，融合更是豪賭 |
| 個人表達 | 你的聲音、你的創意、你去過的地方 = 獨一無二的卡組 |

# 二、召喚系統 — 卡片怎麼來的

## 召喚流程

```
📷 拍攝（取材）→ 🗣️ 咒語（注入靈魂）→ 🌀 煉成（AI 生成）→ 🃏 卡片誕生
```

## 第一步：📷 拍攝 — 取材

- 用手機相機拍攝任何東西，拍到的東西決定卡片的**素材屬性**
- AI 視覺辨識分析畫面內容
- **環境也算**，天氣、光線、時間都會影響結果：
  - 🌧️ 雨天拍 → 卡片附帶「水屬性」加成
  - 🌙 凌晨 3 點拍 → 觸發「暗夜稀有」機率倍增
  - 📍 GPS 地標 → 特定地點出限定卡（例：在東京鐵塔拍 → 有機率出「東京守護神」）

## 第二步：🗣️ 咒語 — 注入靈魂

**不是打字輸入咒語，是真的要「念」出來。**

- 用手機麥克風錄製咒語（3~10 秒）
- AI 分析三個維度：
  - 📝 **語義**：你說了什麼 → 決定卡片技能方向
  - 🎭 **情感**：你的語氣 → 憤怒 = 攻擊卡，溫柔 = 治癒卡，搞笑 = 混沌卡
  - 🔊 **聲紋**：每個人的聲音獨一無二 → 影響隱藏屬性
- 咒語越中二、越離譜、越有創意 → 卡片品質越高
- **咒語會被記錄在卡片上，其他玩家可以回放**

## 第三步：🌀 AI 煉成

- 多模態 AI 綜合所有資訊：
  - 相機畫面 → 卡面視覺主體
  - 咒語語義 → 技能名稱與效果
  - 情感 + 聲紋 → 稀有度 & 隱藏屬性
  - 時空因素 → 附加 Buff
- 每張卡都是獨一無二的存在

## 第四步：🃏 卡片誕生

每張卡片包含：

```
┌──────────────────────────────────┐
│  [卡面圖像 — AI 生成]             │
│                                   │
│  名稱：炎貓                       │
│  等級：Lv.72 ⭐⭐⭐⭐              │
│  被動技能：不屈之焰                │
│  「當等級比對手低時，咒語分 +5」    │
│                                   │
│  召喚者：玩家xxx                   │
│  召喚地點：台北市                  │
│  [▶️ 播放原始咒語]                 │
└──────────────────────────────────┘
```

## 卡片品質（召喚時決定初始等級範圍）

| 品質 | 機率 | 初始等級範圍 | 觸發條件 |
|------|------|-------------|---------|
| ⚪ 普通 | 60% | Lv.1~20 | 隨便拍隨便念 |
| 🟢 精良 | 25% | Lv.15~40 | 畫面清晰 + 咒語有意義 |
| 🔵 稀有 | 10% | Lv.30~60 | 有創意的組合 |
| 🟣 史詩 | 4% | Lv.50~80 | 特殊時間 / 地點 + 情感充沛 |
| 🟡 傳說 | 0.9% | Lv.70~95 | 天時地利人和 |
| 🔴 ??? | 0.1% | Lv.90~100 | 無人知道觸發條件（玩家社群自行挖掘） |

## 每日免費召喚次數

- 免費 5 次 / 天（足夠休閒玩家）
- 對戰累積 5 勝可額外獲得 1 次

# 三、卡片養成系統

## 等級系統

```
⭐          Lv.1 ~ 20
⭐⭐        Lv.21 ~ 40
⭐⭐⭐      Lv.41 ~ 60
⭐⭐⭐⭐    Lv.61 ~ 80
⭐⭐⭐⭐⭐  Lv.81 ~ 99
👑          Lv.100（等級上限）
```

## 升級方式

| 方式 | 等級提升 | 說明 |
|------|---------|------|
| 對戰獲勝 | +1 ~ +3 | 最穩定，打越多越強 |
| 每日餵養 | +2 ~ +5 | 拍指定素材「餵」給卡片 |
| 融合低等級卡 | +5 ~ +10 | 吃掉其他卡來升級（被吃的卡消失） |
| 觸發「以下犯上」 | +5 | 用低等級卡打贏高等級卡的獎勵 |
| 對戰落敗 | +1 | 打了就有成長，不會白費 |

## 餵養系統

```
你的炎貓 Lv.72 今天餓了！

系統提示：「炎貓想吃 🔥 火屬性的東西」

你拍了打火機 → 餵給炎貓 → +3 等級 → Lv.75！
你拍了冰塊   → 炎貓不喜歡 → +0
你拍了火鍋   → 炎貓超愛 → +5！→ Lv.77！
```

每天拍東西餵卡片 → 養成感 → 每天都有理由打開遊戲。

## 融合系統

- 兩張卡可以融合成新卡
- 融合時需要念新咒語（決定融合方向）
- 例：「貓」卡 + 「火焰」卡 + 念「燃燒吧小貓咪」→「炎貓」
- 融合有失敗率 → 失敗會降級其中一張 → 開盲盒心理
- 失敗的「殘骸」可以當材料 → 永遠不會完全浪費

# 四、戰鬥系統

## 核心公式

```
戰鬥力 = 咒語分數 × 等級倍率
```

**數字大的贏。就這一條規則。**

## 等級倍率

| 等級區間 | 倍率 | 意義 |
|---------|------|------|
| ⭐ Lv.1~20 | ×1.0 | 咒語拿幾分就是幾分 |
| ⭐⭐ Lv.21~40 | ×1.3 | 同樣的咒語多 30% 戰鬥力 |
| ⭐⭐⭐ Lv.41~60 | ×1.6 | 開始有明顯差距 |
| ⭐⭐⭐⭐ Lv.61~80 | ×2.0 | 咒語威力翻倍 |
| ⭐⭐⭐⭐⭐ Lv.81~99 | ×2.5 | 高等卡的統治力 |
| 👑 Lv.100 | ×3.0 | 滿等傳說，三倍威力 |

## 等級差距的意義

```
等級高 = 杯子大 → 好的咒語能倒進更多水
等級低 = 杯子小 → 咒語再好也有上限

但等級高不喊 = 空杯子 = 廢物
等級低猛喊   = 小杯裝滿 = 依然有一戰之力
```

低等級打贏高等級的可能性：

| 等級差 | 低等級方需要怎樣才能贏 |
|-------|---------------------|
| 差 10 級 | 咒語比對手好一點就行，常見翻盤 |
| 差 20 級 | 咒語要明顯比對手好，有機會但要拼 |
| 差 30 級 | 你要接近滿分對手要很爛，偶爾發生 |
| 差 40 級 | 你要滿分對手要擺爛，罕見但會上熱搜 |
| 差 50+ 級 | 幾乎不可能，除非對手完全不喊 |

## 咒語評分系統（AI 即時評分 0~30）

| 維度 | 滿分 | 評分標準 |
|------|------|---------|
| 🎯 契合度 | 10 | 咒語內容跟卡片有多相關 |
| 🔥 氣勢 | 10 | 聲音的能量感、情緒張力 |
| 🎪 創意 | 10 | 說法獨特、有趣、出乎意料 |

### 評分透明化

每個維度旁邊都有**一句話解釋為什麼**：

```
🎯 契合度：8/10
   「提到了貓的特徵（肉球），和卡片高度相關」

🔥 氣勢：7/10
   「聲音有力度，但後半段稍弱」

🎪 創意：6/10
   「肉球拍爆的意象不錯，但整體句式常見」
```

### 防刷分機制

```
第 1 次喊「毀滅吧」 → 創意分 7
第 2 次喊「毀滅吧」 → 創意分 4（偵測到重複）
第 3 次喊「毀滅吧」 → 創意分 1（嚴重重複）

AI 記錄你最近 50 場的咒語，重複率越高創意分越低
逼著你每次都想新的說法
```

### 觀眾修正機制

```
在有觀眾的對戰中：

AI 評分後，觀眾有 2 秒可以投票：
  👍「我覺得這句被低估了」
  👎「我覺得這句被高估了」

超過 70% 觀眾按 👍 → 咒語分 +2
超過 70% 觀眾按 👎 → 咒語分 -2

修正幅度小（±2），不會被惡意操控
```

## 三種咒語輸入模式

| 模式 | 操作方式 | 咒語分上限 | 適用場景 |
|------|---------|-----------|---------|
| 🗣️ 語音模式 | 對著手機喊 | 30 | 完整體驗，最強 |
| ⌨️ 文字模式 | 打字輸入 | 25（氣勢上限 5） | 安靜環境、社恐 |
| 🤖 自動模式 | 什麼都不做 | 固定 11 | 純看卡片打架 |

模式切換方式：
- 開口說話 → 自動進入語音模式
- 開始打字 → 自動進入文字模式
- 什麼都不做 → 5 秒後自動模式
- 不需要事先設定，系統即時偵測

## 被動技能系統

每張卡有一個被動技能，翻卡瞬間自動顯示效果，玩家不需要記。

### 6 種被動模板

**❶ 自身增幅型**
```
「當 [條件] 時，我的倍率 +X」

範例：🍜 泡麵 — 深夜食堂
→ 晚上 11 點 ~ 凌晨 3 點對戰時，倍率 +0.5
```

**❷ 自身咒語增強型**
```
「當 [條件] 時，我的咒語分 +X」

範例：🐱 炎貓 — 不屈之焰
→ 當等級比對手低時，咒語分 +5
```

**❸ 削弱對手型**
```
「當 [條件] 時，對手咒語分 -X」

範例：🦊 九尾狐 — 魅惑
→ 對手氣勢分上限變成 5
```

**❹ 底線保護型**
```
「我的戰鬥力不會低於 X」

範例：🛡️ 石牆 — 堅不可摧
→ 戰鬥力不會低於 40
```

**❺ 賭博型**
```
「X% 機率觸發 [強力效果]，失敗則 [懲罰]」

範例：🎰 老虎機 — All in
→ 50% 機率倍率 ×2，50% 機率倍率 ×0.5
→ 翻卡時立即揭曉結果
```

**❻ 連動型**
```
「如果上一回合 [贏了/輸了]，這回合 [效果]」

範例：⚡ 雷神 — 雷鳴連擊
→ 如果上一回合贏了，這回合咒語分 +8
```

被動技能由 AI 根據卡片內容自動分配（拍到貓 → 敏捷類，拍到牆 → 防禦類）

# 五、對戰流程

## 標準對戰：BO3（三局兩勝，90 秒內結束）

```
📦 準備階段（15 秒）
│
│  系統從你的卡組隨機抽 5 張
│  你從中選 3 張 + 排好出場順序
│  順序鎖定後不能改（策略：猜對手怎麼排）
│
↓
🥊 第一回合（20 秒）
│
│  ❶ 翻卡（2 秒）
│     雙方第一張卡同時翻開
│     顯示卡面、等級、被動技能效果
│
│  ❷ 讀秒（3 秒）
│     畫面顯示雙方卡片對峙
│     倒數 3... 2... 1...
│
│  ❸ 咒語時間（5 秒）
│     雙方同時對手機喊咒語
│     畫面分割左右，即時顯示聲波動畫
│
│  ❹ AI 評分（3 秒）
│     三個維度分數一個一個跳出來
│     每個分數附帶一句話解釋
│
│  ❺ 結算（2 秒）
│     戰鬥力數字碰撞動畫
│     贏的一方卡片把輸的彈飛
│
│  → 勝方亮燈 ✅
│
↓
🥊 第二回合（20 秒）
│  流程同上
│  如果已經有人 2:0 → 直接跳到結算
│
↓
🥊 第三回合（20 秒）— 決勝局
│  流程同上
│
↓
🏆 對戰結算（10 秒）
│
│  顯示最終比分（2:0 或 2:1）
│  頒發本場特殊事件
│  播放「本場最佳咒語」回放
│  → [再來一局] [分享戰報] [返回]
```

## 平手處理

當雙方戰鬥力完全相同：

```
⚡ 延長賽：一句定勝負

雙方各有 3 秒說「一句話」
只比 🎪 創意分
分高的贏

如果又平手 → 先喊完的人贏（反應速度決勝）
```

## 特殊事件

對戰中自動觸發，不需要玩家操作：

| 事件 | 觸發條件 | 效果 |
|------|---------|------|
| 🔥 以下犯上 | 等級低 15+ 的卡打贏高等級卡 | 全螢幕特效 + 自動生成短影片 + 卡片 +5 經驗 |
| 💀 輾壓 | 戰鬥力差距 ≥ 30 | 贏的卡片把輸的撞出螢幕外 |
| 🤡 翻車 | 等級高 20+ 卻輸了 | 輸家卡片出現小丑面具，觀眾刷 🤡 |
| ⚡ 險勝 | 戰鬥力差距 ≤ 1 | 慢動作回放 + 心跳音效 |
| 👑 完封 | 3:0 獲勝 | 勝方獲得特殊頭銜框 |
| 🎤 咒語封神 | 單回合咒語分 28+ | 該咒語自動上傳「名咒殿堂」排行榜 |
| 😱 沉默是金 | 某方咒語分 0（完全沒出聲） | 對方卡片露出問號表情，全場尷尬特效 |

## 為什麼戰鬥不會無聊

| 層面 | 設計 |
|------|------|
| 翻卡瞬間 | 不知道對手出什麼 → 每次翻卡都有驚喜 |
| 被動技能 | 可能瞬間改變局勢 → 永遠有意外 |
| 咒語本身 | 你不可能每次喊一樣的話 → 你自己就是隨機元素 |
| 對手的咒語 | 聽到對手喊的話 → 不管輸贏都好笑 |
| 三回合結構 | 起承轉合的敘事弧線 → 每場都是一個故事 |
| 賽季輪換 | 每兩週新規則 → 永遠有新鮮感 |

# 六、對戰模式

## 模式 1：排位賽

```
規則：標準 BO3
配對：同段位玩家
獎勵：段位分數
```

### 段位表

| 段位 | 分數 |
|------|------|
| 🐣 見習咒術師 | 0 ~ 99 |
| 📖 學徒咒術師 | 100 ~ 299 |
| 🔮 正式咒術師 | 300 ~ 599 |
| ⚡ 高階咒術師 | 600 ~ 999 |
| 🌙 大咒術師 | 1000 ~ 1499 |
| 👑 傳說咒術師 | 1500+ |
| 💀 禁咒者 | 全服第 1 名（只有一人） |

勝 → +15~25 分（以下犯上額外 +10）
敗 → -10~20 分（翻車額外 -5）

## 模式 2：開放賽

```
規則：標準 BO3
配對：完全隨機，無視段位和等級
獎勵：金幣 + 經驗值（不影響段位）
特色：Lv.15 新手 vs Lv.95 老玩家 → 奇蹟發生的地方
      開放賽的「以下犯上」會推播到全服動態牆
```

## 模式 3：好友對決

```
規則：標準 BO3
配對：邀請指定好友
獎勵：無（純粹好玩）
特色：
  - 可開語音頻道邊打邊垃圾話
  - 可設觀戰模式讓其他好友看
  - 觀戰者可以發彈幕影響比賽：
    😂 ×50 → 「全場爆笑」→ 雙方創意分 +2
    🔥 ×50 → 「觀眾狂熱」→ 雙方氣勢分 +2
    💀 ×50 → 「社會性死亡」→ 被針對方咒語分 -3
```

## 模式 4：單卡決鬥

```
規則：一回合、一張卡、一句咒語
配對：隨機
時間：30 秒一場
特色：最速對戰，排隊、坐車、無聊時隨手一場
      門檻最低，適合新手入門，也適合老手練咒語
```

## 模式 5：世界 Boss（每週一次）

```
規則：全服合力打一隻 AI 控制的超強 Boss
流程：
  Boss 血量 = 1,000,000
  每個玩家每天可以挑戰一次
  出一張卡 + 喊一句咒語 → 戰鬥力 = 對 Boss 的傷害
  全服累積傷害直到 Boss 被打倒

獎勵：
  Boss 被打倒 → 全服每人獲得獎勵寶箱
  傷害排行前 100 → 限定卡面邊框
  傷害排行第 1 → 限定稱號「屠龍者」
```

## 模式 6：賽季挑戰賽（每兩週輪換）

| 挑戰 | 規則 | 迷因效果 |
|------|------|---------|
| 🤫 低語之戰 | 必須用氣音悄悄話念咒語 | 一群人對手機 ASMR |

# 七、社交與傳播系統

## 咒語圖鑑 — 病毒傳播核心

- 每張卡都保存了創建時的咒語錄音
- 玩家可瀏覽「全服最離譜咒語排行榜」
- 對戰中的咒語自動存檔，可回放

## 自動生成短影片

每場對戰自動剪輯成 15 秒影片：

```
0-2 秒    雙方卡面展示
2-4 秒    翻卡動畫
4-8 秒    分屏畫面：左邊喊、右邊喊
8-12 秒   AI 評分彈出（帶特效）
12-15 秒  結算動畫 + 勝者慶祝
```

一鍵分享到 TikTok / IG Reels / 小紅書 / YouTube Shorts

## 共鳴召喚 — 線下社交

- 兩個玩家面對面，同時拍同一個東西 + 合念咒語
- 產出的卡片比單人強得多
- 卡面上會有兩人的名字
- 線下見面的理由 → 社群活動自然形成

## 每日現實任務

系統每天發布任務，引導玩家探索現實世界：

- 「今日素材：拍一棵樹」→ 全服用各種樹 + 各種咒語比誰強
- 「聲音挑戰：用唱歌的方式念咒語」→ 社群瘋傳
- 「地點獵人：去一個你從沒去過的地方拍照召喚」

## AR 卡片展示

- 用 AR 把自己的卡片在現實中展示
- 卡片會動（AI 生成的短動畫）
- 可以把卡片「放」在現實場景中拍照分享
